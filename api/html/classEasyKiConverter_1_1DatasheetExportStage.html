数据手册导出阶段

管理数据手册(Datasheet)的并行导出任务。 每个元器件的数据手册导出由DatasheetExportWorker执行。

目录结构: <outputPath>/ └── datasheets/ ← 数据手册目录 ├── component1.pdf ├── component2.pdf └── ...

使用 TempFileManager 进行临时写入、备份提交和失败回滚。

并发配置:

  • 最大并发数: 2（PDF文件较大，下载+I/O较慢）
