详细描述

整体导出进度

跟踪整个导出任务的整体进度，包含预加载进度和各导出类型的进度

使用示例:

ExportOverallProgress progress;
progress.currentStage = ExportOverallProgress::Stage::Preloading;
progress.totalComponents = 100;
