详细描述

KiCad符号几何形状的JSON序列化工具

负责以下类型的序列化：

  • SymbolBBox (边界框)
  • SymbolRectangle (矩形)
  • SymbolCircle (圆形)
  • SymbolArc (圆弧)
  • SymbolEllipse (椭圆)
  • SymbolPolyline (多段线)
  • SymbolPolygon (多边形)
  • SymbolPath (路径)
  • SymbolText (文本)
