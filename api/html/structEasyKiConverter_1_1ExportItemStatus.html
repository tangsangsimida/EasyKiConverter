详细描述

单个导出项的状态信息

跟踪单个元器件的导出状态，包括进度、错误信息、文件路径等

状态流转: Pending -> InProgress -> Success/Failed/Skipped

使用示例:

ExportItemStatus status;
status.status = ExportItemStatus::Status::InProgress;
status.startTime = QDateTime::currentDateTime();
