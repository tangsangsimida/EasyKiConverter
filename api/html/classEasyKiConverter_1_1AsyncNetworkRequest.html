详细描述

异步网络请求，支持取消

支持：

  • 通过 cancel() 取消请求
  • 进度跟踪
  • 自动重试与指数退避
  • Gzip 解压
  • 带请求体的 POST 请求

使用示例： auto* request = client->getAsync(url, policy); connect(request, &AsyncNetworkRequest::finished, this, [](const NetworkResult& result) { if (result.wasCancelled) { ... } }); // 取消请求：request->cancel();
