详细描述

单个导出类型的总体进度

跟踪一种导出类型（如符号导出）的整体进度，包含该类型下所有元器件的状态

使用示例:

ExportTypeProgress progress;
progress.typeName = "Symbol";
progress.totalCount = 100;
progress.completedCount = 50;
