预览图导出阶段

管理预览图(Preview Images)的并行导出任务。 每个元器件的预览图导出由PreviewImagesExportWorker执行。

目录结构: <outputPath>/ └── previews/ ← 预览图目录 ├── component1_preview_1.png ├── component1_preview_2.png └── ...

使用 TempFileManager 进行临时写入、备份提交和失败回滚。

并发配置:

  • 最大并发数: 4（预览图导出是图像处理+I/O密集型）
