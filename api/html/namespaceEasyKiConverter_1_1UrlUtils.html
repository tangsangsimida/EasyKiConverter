验证 URL 是否在允许的白名单内

检查 URL 的 scheme（仅允许 https）和 host（按 ResourceType 限制）。 用于在 NetworkClient 发送请求前统一校验，防止访问非预期主机。

参数

resourceType资源类型，用于确定允许的 host 集合
errorMessage输出参数，校验失败时的错误信息
返回
如果 URL 合法则返回 true
