RAII 辅助类：确保状态对象在析构时（或显式调用时）清理数据

使用场景：

  • 在 Worker 析构时自动清理
  • 在异常处理中确保清理

注意：此 RAII 类会在析构时清理数据，不要在数据仍需使用时析构此对象
