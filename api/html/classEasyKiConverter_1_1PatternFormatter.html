模式格式化器

支持占位符的日志格式化器：

  • %{timestamp} - 时间戳 (2024-01-15 10:30:45.123)
  • %{time} - 时间 (10:30:45.123)
  • %{date} - 日期 (2024-01-15)
  • %{level} - 日志级别 (DEBUG/WARN/ERROR)
  • %{module} - 模块名 (Network/Export)
  • %{thread} - 线程 ID
  • %{threadname} - 线程名称
  • %{file} - 文件名（含路径）
  • %{shortfile} - 简化文件名（不含路径）
  • %{line} - 行号
  • %{function} - 函数名
  • %{message} - 日志消息
  • %{newline} - 换行符
  • %% - 百分号

默认模式: "[%{timestamp}] [%{level}] [%{module}] [%{thread}] %{message} (%{shortfile}:%{line})"
