void EasyKiConverter::FootprintExportWorker::run ( )
overridevirtual

执行封装导出任务

在线程池中被调用，执行实际的导出逻辑:

  1. 检查封装数据是否可用
  2. 创建输出目录
  3. 检查是否需要覆盖已有文件
  4. 执行导出并写入.kicad_mod文件
  5. 发射completed信号

实现了 EasyKiConverter::IExportWorker.
