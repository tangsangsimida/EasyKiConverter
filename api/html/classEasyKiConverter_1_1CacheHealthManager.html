详细描述

缓存自愈管理器

负责检测和修复损坏的缓存数据：

  • 验证缓存目录的元数据完整性
  • 清理无效/损坏的CAD数据文件
  • 清理空的或损坏的预览图
  • 迁移遗留格式的数据手册文件
  • 清理损坏的3D模型缓存
