详细描述

控制台日志输出器

支持彩色输出的控制台 Appender，根据日志级别自动着色：

  • Trace: 灰色
  • Debug: 青色
  • Info: 白色（默认）
  • Warn: 黄色
  • Error: 红色
  • Fatal: 红底白字

支持异步输出模式，减少对主线程的性能影响。
