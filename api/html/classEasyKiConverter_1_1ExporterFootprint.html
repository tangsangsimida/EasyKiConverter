批量导出封装库（KiCad .kicad_mod 格式）

参数
footprints封装列表
libName库名称（用于构建 3D 模型相对路径）
filePath输出目录路径（.pretty 目录）
preferWrl是否优先使用 WRL 格式（默认 true，向后兼容）
exportStep是否同时导出 STEP 格式（默认 false，向后兼容）
useAbsolutePaths3D模型路径是否使用绝对路径（默认 false，使用相对路径）
注解
默认参数仅导出 WRL，以保持与旧版本的兼容行为。 调用方应显式传入 exportStep=true 以启用双格式导出。
返回
bool 是否成功
