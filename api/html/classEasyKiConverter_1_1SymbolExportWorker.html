详细描述

符号导出Worker

在线程池中执行单个元器件的符号导出任务。 负责将元器件的符号数据转换为KiCad符号格式(.kicad_sym)并写入磁盘。

使用QThreadPool时，Worker通过QRunnable接口执行。
