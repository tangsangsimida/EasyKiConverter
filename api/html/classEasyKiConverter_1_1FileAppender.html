详细描述

文件日志输出器

支持以下特性：

  • 按大小滚动（默认 10MB）
  • 保留历史文件数量限制
  • 异步写入（后台线程）
  • 缓冲刷新
  • 路径安全验证
  • 有界队列防止内存溢出
