# 窗口状态手动测试清单

本文档用于验证主窗口启动、显示、最大化恢复、窗口化恢复和配置持久化行为，重点覆盖 2026-04-19 修复的窗口启动映射与状态恢复回归问题。

## 适用范围

- Linux 桌面环境
- Qt Quick `ApplicationWindow`
- 使用 `Qt.FramelessWindowHint` 的主窗口

如条件允许，建议分别在以下环境执行：
- X11
- Wayland

## 测试前准备

1. 使用最新代码构建应用：

```bash
python3 tools/python/build_project.py -j 4
```

2. 确认当前配置文件存在：

```bash
~/.config/EasyKiConverter/EasyKiConverter/config.json
```

3. 测试期间如需重置窗口状态，可删除配置中的以下字段或直接备份后删除配置文件：
- `windowX`
- `windowY`
- `windowWidth`
- `windowHeight`
- `windowMaximized`

## 用例 1：首次启动应显示主窗口

**目的**：
验证应用启动后主窗口能真正映射到桌面，而不是仅进程运行但窗口不可见。

**步骤**：
1. 启动应用：

```bash
./build/bin/easykiconverter
```

2. 观察桌面、任务栏和 Alt+Tab 列表。

**预期结果**：
- 主窗口可见
- 任务栏存在窗口条目
- Alt+Tab 可见窗口
- 应用不会只在后台运行而不显示窗口

## 用例 2：普通窗口关闭后应恢复原尺寸和位置

**目的**：

**预期结果**：
