# 隐私政策

**最后更新时间：** 2026年3月13日

## 1. 引言
EasyKiConverter（"我们"、"我们方"或"我们的"）致力于保护您的隐私。本隐私政策解释了当您使用我们的桌面应用程序时，我们如何处理信息。

## 2. 数据收集
**我们不收集、存储、共享或传输任何个人数据。**

- **仅限本地偏好设置**：应用程序仅将某些用户偏好设置（如窗口位置、主题设置或上次使用的选项）严格保存在您的本地设备上。这些文件存储在本地，绝不会传输给我们或任何第三方。
- **无个人信息**：我们不收集姓名、电子邮件地址、位置数据、设备标识符或任何其他可识别个人身份的信息。
- **无分析工具**：我们不使用任何第三方分析或广告服务。

## 3. 互联网使用
应用程序可能仅根据用户输入的元件编号访问互联网，从 LCSC（嘉立创）服务器获取公共元件预览图像。在此过程中不发送任何特定于用户的数据。

## 4. 第三方服务
我们的应用程序是根据 GPL-3.0 许可证的开源软件。我们不集成任何收集用户数据的第三方 SDK。

## 5. 政策变更
我们可能会不时更新本隐私政策。任何更改都将发布在本页面上。
