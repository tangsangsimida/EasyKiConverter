# 📋 System Requirements

## 🌟 Supported Platforms
- **Windows**: Windows 10 or later
- **Linux**: Ubuntu 20.04 or later
- **macOS**: macOS 10.15 (Catalina) or later

## 🔧 Required Software
- **Compiler**:
  - GCC >= 7.0 (Linux/macOS)
  - MSVC >= 2019 (Windows)
- **Build System**: CMake >= 3.10
- **Qt Version**: Qt5
- **Other Dependencies**: zlib, curl

## 📦 Supported Component Types
- **Symbols**: KiCad symbol format
- **Footprints**: KiCad footprint format
- **3D Models**: KiCad 3D model format
