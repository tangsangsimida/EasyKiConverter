# EasyKiConverter 性能基准基线 (Performance Baseline)

本文件用于记录项目各版本的核心性能基准。CI 会根据此基线进行回归验证。

## 核心转换性能指标

| 测试项 | 基准耗时 (ms) | 允许偏差 (%) | 说明 |
| :--- | :--- | :--- | :--- |
| 单个 LCSC 封装解析 (简单元件) | 50 | 15 | 如 0603 电阻 |
| 单个 LCSC 封装解析 (复杂 IC) | 150 | 20 | 如 LQFP-100 封装 |
| BOM 表完整转换 (50 行以内) | 1000 | 15 | 包含网络采集和本地生成 |
| KiCad 库文件序列化 (100+ 元件) | 300 | 10 | 存盘性能 |

## 验证环境参考
- **OS**: Windows 10/11
- **Qt Version**: 6.6.x
- **Compiler**: MSVC 2019+ / MinGW 11+
- **CPU**: Intel i5-11th Gen / AMD Ryzen 5 5000+ 或等效 CI Runner
