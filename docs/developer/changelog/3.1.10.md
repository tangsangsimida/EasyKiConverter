# 更新日志 — v3.1.10

## [3.1.10] - 2026-06-01

### 新功能

- **响应式 UI 重构** (#174)
  - 添加响应式侧边栏布局及导出控制组件
  - 优化主窗口交互体验

- **缓存写入保护机制**
  - 防止过期数据覆盖新条目

- **临时文件管理** (#172)
  - 引入 TempFileManager 进行安全的临时文件管理
  - 支持备份和提交语义

- **UI 交互优化**
  - 为卡片组件添加悬停缩放延迟定时器功能
  - 添加退出动画效果并优化按钮交互体验

- **测试基础设施**
  - 为网络客户端测试添加 zlib 依赖支持

### 修复

- **导出稳定性** (#172)
  - 修复导出取消崩溃问题
  - 重新设计预览图悬停 UI

- **Windows 部署**
  - 修复 Windows 部署中 ICU shim DLL 问题
  - 添加验证步骤

- **3D 模型** (#168)
  - 修复 3D 模型偏移对齐问题
  - 新增 CLI 数据手册导出支持

### 改进

- **UI 组件**
  - 统一 UI 组件的交互延迟配置
  - 优化滑块对话框基础组件的动画效果
  - 响应式布局优化与 UI 稳定性修复
