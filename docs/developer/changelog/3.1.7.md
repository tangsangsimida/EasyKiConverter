# 更新日志 — v3.1.7

**发布日期**: 2026-05-09

> 详细更新内容请参见 [CHANGELOG.md](../CHANGELOG.md)

## [3.1.7] - 2026-05-09

### 新增

- **3D 模型功能增强**
  - 添加 STEP 模型偏移补偿功能
  - 添加 3D 模型 Z 轴偏移对齐功能
  - CLI 支持 3D 格式选择（`--3d-model-format`）
  - 封装导出归一化处理

- **符号库导出增强**
  - 符号库描述导出功能
  - 元器件描述编辑支持

- **导出重试模式**
  - 为库导出功能实现重试模式，保留已成功导出的组件

### 修复

- **ComponentService 竞态问题**
  - 修复陈旧组件条目导致验证请求被静默跳过的问题
  - 引入 `requestActive` 标志跟踪请求状态
  - 使用 `find()` 替代 `operator[]` 防止异步回调重建已删除条目

- **3D 模型导出**
  - 修复 STEP 偏移计算的缓存依赖问题
  - 修复 3D 模型缓存使用问题（先缓存后网络策略）
  - 修正 STEP 模型 Z 轴偏移补偿逻辑

- **导出错误处理**
  - 增强导出阶段错误处理能力

### 改进

- **缓存优化**
  - 统一 3D 模型坐标单位为毫米
  - 优化 STEP 几何计算性能
  - 优化 3D 模型缓存验证逻辑
  - 预加载阶段加载 OBJ 二进制数据避免重复磁盘 I/O
  - 增强缓存完整性检查（验证文件大小大于 0）

- **足迹导出**
  - 增强足迹导出阶段的 STEP 数据获取逻辑
