# Changelog — v3.1.7

**Release Date**: 2026-05-09

> For detailed changes, see [CHANGELOG.md](../CHANGELOG.md)

## [3.1.7] - 2026-05-09

### Added

- **3D Model Enhancements**
  - Added STEP model offset compensation
  - Added 3D model Z-axis offset alignment
  - CLI support for 3D format selection (`--3d-model-format`)
  - Footprint export normalization

- **Symbol Export Enhancements**
  - Symbol library description export
  - Component description editing support

- **Export Retry Mode**
  - Implemented retry mode for library export, preserving successfully exported components

### Fixed

- **ComponentService Race Condition**
  - Fixed stale fetching entries causing validation requests to be silently skipped
  - Introduced `requestActive` flag to track request state
  - Used `find()` instead of `operator[]` to prevent async callbacks from recreating deleted entries

- **3D Model Export**
  - Fixed STEP offset calculation cache dependency issue
  - Fixed 3D model cache usage (cache-first strategy)
  - Fixed STEP model Z-axis offset compensation logic

- **Export Error Handling**
  - Enhanced export stage error handling

### Improved

- **Cache Optimization**
  - Unified 3D model coordinate units to millimeters
  - Optimized STEP geometry calculation performance
  - Optimized 3D model cache validation logic
  - Preload phase loads OBJ binary data to avoid repeated disk I/O
  - Enhanced cache integrity checks (verify file size > 0)

- **Footprint Export**
  - Enhanced footprint export stage STEP data fetching logic
