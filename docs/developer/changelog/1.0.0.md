## 版本说明

本项目遵循语义化版本规范：MAJOR.MINOR.PATCH

- **MAJOR**：不兼容的 API 更改
- **MINOR**：新功能（向后兼容）
- **PATCH**：Bug 修复（向后兼容）

## 更新类型说明

- **新增**：新功能
- **修复**：Bug 修复
- **更改**：现有功能的更改
- **移除**：移除的功能
- **安全**：安全相关的修复
- **弃用**：即将移除的功能
