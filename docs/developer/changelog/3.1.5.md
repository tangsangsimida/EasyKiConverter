# 更新日志 — v3.1.5

**发布日期**: 2026-04-26

> 详细更新内容请参见 [CHANGELOG.md](../CHANGELOG.md)

## [3.1.5] - 2026-04-26

### 新增

- **测试基础设施**
  - 添加 CadDataLoader 单元测试
  - 添加 ComponentListViewModel 单元测试

- **图形 API 支持**
  - 增加对多种图形 API 的支持：OpenVG、Null、Direct3D12
  - 优先使用 Vulkan 渲染后端并添加渲染 API 检测

- **调试导出功能**
  - 添加 debug 数据导出功能
  - 使用原子写入改进调试数据导出

- **构建配置增强**
  - 支持多平台 Qt 路径配置和构建选项

### 修复

- **ComponentService**
  - 添加元器件请求取消功能
  - 修复 API 业务层错误处理

### 改进

- **UI 样式统一**
  - 使用 AppStyle 统一 QML 组件样式

### 其他

- 移除 Docker 环境构建工作流
- 更新项目文档
