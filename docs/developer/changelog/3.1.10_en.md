## [3.1.10] - 2026-06-01

### New Features

- **Responsive UI Redesign** (#174)
  - Add responsive sidebar layout with export control components
  - Improve main window UX

- **Cache Write-back Protection**
  - Prevent stale data from overwriting fresh entries

- **Temporary File Management** (#172)
  - Introduce TempFileManager for safe temporary file management
  - Support backup and commit semantics

- **UI Interaction Optimization**
  - Add hover scale delay timer for card components
  - Add exit animation effects and optimize button interaction experience

- **Test Infrastructure**
  - Add zlib dependency support for network client tests

### Bug Fixes

- **Export Stability** (#172)

- **Windows Deployment**
  - Fix Windows deployment ICU shim DLL issues
  - Add verification steps

- **3D Model** (#168)
  - Fix 3D model offset alignment issues
  - Add CLI datasheet export support

### Improvements

- **UI Components**

  - Responsive layout optimization and UI stability fixes
