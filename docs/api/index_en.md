# API Reference

This section contains auto-generated API reference documentation for EasyKiConverter, generated from source code comments by [Doxygen](https://www.doxygen.nl/).

## Accessing API Documentation

API documentation is automatically generated by CI workflows and deployed to GitHub Pages:

- **[C++ API Reference](./html/index.html)** - Complete reference for classes, functions, signals and slots

## Local Generation

To generate API documentation locally:

```bash
# Install Doxygen and Graphviz
# Windows: choco install doxygen graphviz
# macOS: brew install doxygen graphviz
# Linux: sudo apt install doxygen graphviz

# Generate documentation
doxygen Doxyfile

# View documentation
# Output directory: docs/api/html/index.html
```

## Documentation Structure
