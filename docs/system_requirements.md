# 📋 系统要求

[English Version](system_requirements_en.md)

## 🌟 支持的平台
- **Windows**: Windows 10 或更高版本
- **Linux**: Ubuntu 20.04 或更高版本
- **macOS**: macOS 10.15 (Catalina) 或更高版本

## 🔧 必要的软件
- **编译器**:
  - GCC >= 7.0（Linux/macOS）
  - MSVC >= 2019（Windows）
- **构建工具**: CMake >= 3.10
- **Qt版本**: Qt5
- **其他依赖**: zlib, curl

## 📦 支持的元件类型
- **符号**: KiCad 符号格式
- **封装**: KiCad 封装格式
- **3D模型**: KiCad 3D 模型格式
